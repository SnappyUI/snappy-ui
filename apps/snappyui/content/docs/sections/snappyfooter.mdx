## Props

| **Name**      | **Type**         | **Default** | **Description**                                                                                          |
| ------------- | ---------------- | ----------- | -------------------------------------------------------------------------------------------------------- |
| `variant`     | `string`         | `'simple'`  | Footer style variant: 'simple', 'detailed', 'minimal', 'centered', 'modern', 'classic', 'elegant' |
| `size`        | `string`         | `'default'` | Footer size: 'sm', 'default', 'lg'                                                                       |
| `logo`        | `ReactNode`      | `-`         | Logo component to display in the footer                                                                  |
| `copyright`   | `string`         | `-`         | Copyright text to display                                                                                |
| `columns`     | `FooterColumn[]` | `[]`        | Navigation columns with links                                                                            |
| `socialLinks` | `FooterLink[]`   | `[]`        | Social media links                                                                                       |
| `bottomText`  | `string`         | `-`         | Additional text displayed at the bottom                                                                  |
| `className`   | `string`         | `-`         | Additional CSS classes for the footer                                                                    |
