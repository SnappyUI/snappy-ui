## Props

| **Name** | **Type** | **Default** | **Required** | **Description** |
|-------------|----------|-------------|--------------|---------------------------------------------------------|
| **name** | string   | -           | Yes          | The name of the person giving the testimonial.          |
| **title** | string   | -           | Yes          | The title or position of the person.                    |
| **quote** | string   | -           | Yes          | The actual testimonial text.                            |
| **avatar** | string   | -           | Yes          | URL or path to the avatar image.                        |
| **alt** | string   | -           | Yes          | Alternative text for the avatar image (accessibility). |
| **className** | string   | -           | No           | Additional CSS classes for the testimonial container.   |

## Props

| Name                 | Type       | Default   | Description                                       |
| -------------------- | ---------- | --------- | ------------------------------------------------- |
| **name**             | string     | -         | Name of the person giving testimonial (required)  |
| **title**            | string     | -         | Job title or position of the person (required)    |
| **quote**            | string     | -         | The testimonial text content (required)           |
| **avatar**           | string     | -         | URL to the avatar image (required)                |
| **alt**              | string     | -         | Alt text for the avatar image (required)          |
| **rating**           | number     | 5         | Star rating out of 5 (optional)                   |
| **className**        | string     | -         | Extra Tailwind/custom classes for container       | testimonial container.   |
