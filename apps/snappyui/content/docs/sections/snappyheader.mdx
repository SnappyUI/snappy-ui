# Snappy Header Component

A flexible header component with multiple variants, sizes, and layouts to create attractive and engaging page headers for your application.

## Basic Usage

<div className="flex flex-col items-center justify-center mb-6 border border-zinc-800 rounded-lg">
  <DefaultHeader />
</div>

```tsx
import { HeaderContainer } from "@/components/ui/snappy-header";

export default function BasicHeader() {
  return (
    <HeaderContainer
      title="Welcome to Our Platform"
      description="The most comprehensive solution for all your needs. Build beautiful interfaces with our component library and tools."
      badge={<span className="px-3 py-1 text-xs font-medium bg-blue-100 text-blue-800 rounded-full">New Feature</span>}
      action={
        <div className="flex gap-4">
          <button className="px-6 py-2 bg-blue-600 text-white rounded-md">Get Started</button>
          <button className="px-6 py-2 bg-transparent border border-gray-300 rounded-md">Learn More</button>
        </div>
      }
    />
  );
}
```

## Custom Background Image

You can add a background image and control the overlay opacity:

```tsx
<HeaderContainer
  title="Header with Background"
  description="Custom background image with overlay."
  backgroundImage="/your-image-path.jpg"
  backgroundOverlay={true}
  overlayOpacity={0.7}
/>
```

## Badges and Actions

Add badges or action buttons to your header:

```tsx
<HeaderContainer
  title="Header with Badge and Actions"
  description="Includes a badge and action buttons."
  badge={<span className="px-3 py-1 text-xs font-medium bg-blue-100 text-blue-800 rounded-full">New</span>}
  action={
    <div className="flex gap-4">
      <button className="px-6 py-2 bg-blue-600 text-white rounded-md">Primary Action</button>
      <button className="px-6 py-2 bg-transparent border border-gray-300 rounded-md">Secondary Action</button>
    </div>
  }
/>
```

## Animation Controls

Toggle animations on or off:

```tsx
// With animations (default)
<HeaderContainer
  title="Animated Header"
  description="Header with entrance animations."
  animated={true}
/>

// Without animations
<HeaderContainer
  title="Static Header"
  description="Header without animations."
  animated={false}
/>
```

## Props

| Name                 | Type       | Default   | Description                                       |
| -------------------- | ---------- | --------- | ------------------------------------------------- |
| **title**            | ReactNode  | -         | Header title (required)                           |
| **description**      | ReactNode  | -         | Optional description text                         |
| **variant**          | string     | 'default' | Header style variant                              |
| **size**             | string     | 'default' | Header size (sm, default, lg, xl)                 |
| **alignment**        | string     | 'left'    | Content alignment (left, center, right)           |
| **className**        | string     | -         | Extra Tailwind/custom classes for container       |
| **titleClassName**   | string     | -         | Extra classes for title element                   |
| **descriptionClassName** | string | -         | Extra classes for description element             |
| **backgroundImage**  | string     | -         | Optional background image URL                     |
| **backgroundOverlay**| boolean    | true      | Show dark overlay over background image           |
| **overlayOpacity**   | number     | 0.5       | Opacity of overlay (0-1)                          |
| **animated**         | boolean    | true      | Enable/disable entrance animations                |
| **action**           | ReactNode  | -         | Optional action element(s) (buttons, forms, etc.) |
| **badge**            | ReactNode  | -         | Optional badge element                            |
