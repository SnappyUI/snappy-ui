## Props

### ClipboardButton Component

| **Name**      | **Type**                                     | **Default** | **Description**                           |
| ------------- | -------------------------------------------- | ----------- | ----------------------------------------- |
| **value**     | string                                       | -           | The text value to copy                    |
| **onCopied**  | () => void                                   | -           | Optional callback when copy is successful |
| **className** | string                                       | -           | Optional additional class names           |
| **variant**   | "default" \| "outline" \| "ghost"            | "outline"   | Button style variant                      |
| **...props**  | React.ButtonHTMLAttributes\<HTMLButtonElement\> | -        | Any other button props                    |

### ClipboardCopy Component

| **Name**          | **Type**    | **Default** | **Description**                                     |
| ----------------- | ----------- | ----------- | --------------------------------------------------- |
| **value**         | string      | -           | The full text value to be copied                    |
| **preview**       | boolean     | true        | Whether to show a truncated preview                 |
| **previewLength** | number      | 40          | Max length for the preview                          |
| **className**     | string      | -           | Optional additional class names for the container   |
| **onCopied**      | () => void  | -           | Optional callback when copy is successful           |
