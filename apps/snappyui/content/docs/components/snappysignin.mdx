## Basic Usage

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <div className="flex flex-col items-center justify-center mb-6 rounded-lg py-5 px-5">
      <SnappySignIn />
    </div>
  </Tab>

  <Tab value="Code">
    ```tsx
    import {
    LoginPage,
    LoginHeader,
    EmailInput,
    PasswordInput,
    ForgotPasswordLink,
    SubmitButton,
    SignUpLink
    } from "@/components/ui/snappy-login";

    <LoginPage  >
    <LoginHeader title="Welcome Back" subtitle="Log in to your account" />
    <EmailInput />
    <PasswordInput />
    <ForgotPasswordLink />
    <SubmitButton onClick={(data) => {
                return new Promise(resolve => {
                    setTimeout(() => {
                        console.log("I will get this data while submitting", data);
                        resolve(true); // This is important
                    }, 5000);
                });
    }} />
    <SignUpLink />
    </LoginPage>
    ```
  </Tab>
</Tabs>

## Props

### LoginPage
|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|
| **children**     |`ReactNode`| `-` | Contents of the component |
| **className**     |`string`| `-` | Tailwind utility classes |

### Login Header

|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|
| title | string | "Sign In" | Main heading text |  
subtitle | string | "Enter your credentials to access your account" | Subheading text |
logo | ReactNode | null | Optional logo element | 
children | ReactNode | - | Additional elements inside header | 
 className | string | - | Tailwind utility classes |

### EmailInput

|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|
label | string | "Email" | Label text |
placeholder | string | "you@example.com" | Input placeholder |
 className | string | - | Tailwind utility classes |

### PasswordInput

|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|
| label | string | "Password" | Label text | 
placeholder | string | "••••••••" | Input placeholder | 
showToggle | boolean | true | Show/hide password toggle button | 
className | string | - | Tailwind utility classes |

### ForgotPasswordLink

|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|

className | string | - | Tailwind utility classes |

### SubmitButton

|<b>Name</b> |	<b>Type</b> | <b>Default</b> |	<b>Description</b> |
| ------------- | -------------------------------------------------------------------------------------------------- | ---------------- | -----------------------------|
