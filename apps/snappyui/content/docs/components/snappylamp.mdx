## Props use main container of Lamp

| Name                | Type       | Default    | Description                                      |
| ------------------- | ---------- | ---------- | ------------------------------------------------ |
| **variant**         | string     | 'default'  | Lamp style variant                               |
| **size**            | string     | 'default'  | Lamp size (sm, default, lg, xl)                  |
| **children**        | ReactNode  | -          | Lamp content                                     |
| **className**       | string     | -          | Extra Tailwind/custom classes (optional)         |
| **spotlightColor**  | string     | '#0ea5e9'  | Color of the main spotlight                      |
| **glowColor**       | string     | '#0ea5e9'  | Color of the glow effect                         |
| **backgroundColor** | string     | 'black'    | Background color (for default variant)           |
| **backgroundOpacity** | number   | 0.95       | Background opacity (0-1)                         |
| **glowOpacity**     | number     | 0.3        | Opacity of the glow effect (0-1)                 |
| **borderRadius**    | string     | '1.75rem'  | Border radius for lamp container                 |
| **intensity**       | number     | 20         | Intensity of glow effect (blur amount in px)     |
| **animated**        | boolean    | true       | Enable/disable animation effects                 |

---

## Lamp Variants

| Variant      | Description                                       |
| ------------ | ------------------------------------------------- |
| **default**  | Standard lamp with basic spotlight                |
| **spotlight**| Enhanced spotlight effect with softer background  |
| **gradient** | Gradient background with spotlight effect         |
| **soft**     | Soft diffused light effect                        |
| **neon**     | Vibrant glow effects with enhanced intensity      |
