# Snappy Card Component

A flexible card component that supports images, titles, and custom content with various styling variants.

## Props

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **title**     | string      | -              | The title of the card                         |
| **className** | string      | -              | Additional CSS classes                        |
| **content**   | string      | -              | The main content of the card                  |
| **image**     | string      | -              | URL of the card's image                       |
| **variant**   | string      | 'default'      | The style variant of the card                 |
| **children**  | ReactNode   | -              | Allows passing React nodes into the component |
