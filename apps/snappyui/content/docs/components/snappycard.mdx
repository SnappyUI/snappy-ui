## Props

### Horizontal/Blog Card

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **className**  | string   | -              | 	Tailwind utility classes |
| **imageUrl**  | string   | -              | 	Source URL of the image |
| **imageAlt**  | string   | -              | 	Alt text for the image |
| **title**  | string   | -              | 	Title text for the card |
| **description	**  | string   | -              | 	Description text |
| **text	**  | string   | -              | 	Button label text |
| **onClick**  | cb   | -              | 	Callback function for button click |

### Profile Card

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **variant**     | default,elevated,bordered,minimal      | default              | Visual style of the card                     |
| **className** | string      | -              | Additional CSS classes                        |
| **hoverEffect**   | boolean      | false              | Enable hover animation                  |
| **centered**     | boolean      | true              | Center content(card body)                      |
| **imageUrl**   | string      | 'default'      | Image source URL  |
| **imageAlt**  | string   | -              | 	Alt text for the image |
| **size**  | small,medium,large   | medium              | 	Image size |
| **bordered**  | boolean   | true              | 	Show border ring |
| **icon**  | ReactNode   | -              | 	Icon to display |
| **color**  | "blue" , "green" , "red" , "yellow" , "purple"   | "blue"          | 	Badge color |
| **name**  | string   | -              | 	Name to display |
| **designation**  | string   | -              | 	Job title or position |
| **bio**  | string   | -              | 	Short biography text |
| **stats**  |label: string, value: string , number []   | -              | 	Array of stats |
| **icon**  | ReactNode   | -              | 		Icon component |
| **href**  | string   | -              | 		URL for social link |
| **label**  | string   | -              | 	ARIA label for accessibility |
| **email**  | string   | -              | 	Email address |
| **phone**  | string   | -              | 	Phone number |
| **location**  | string   | -              | 	User's location |
| **name**  | string   | -              | 	Skill name |
| **level**  | number   | -              | 	Skill level (0-100) |
| **skills**  | name: string, level: number []   | -              | 	Array of skill objects |
