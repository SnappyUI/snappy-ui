## Props

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **words**     | string      | -              | Text which will be animated                         |
| **className** | string      | -              | Additional CSS classes                        |
| **typingSpeed**   | float      | -              | Speed for the amnimation                 |
| **direction**     | string      | -              | choose from "up",  "down", "left", "right"                       |
| **staggerTime**   | float      | -      | the delay between the animation of each element in a sequence |
| **fromColor**  | rgb(color)   | -              | color for transitions |
| **toColor**  | rgb(color)   | -              | color for transitions |
| **rotation**  | boolean   | false           | True or false |
