### Props

I've analyzed the JsonViewer component and will create a comprehensive prop table for it. The component has two main interfaces: JsonViewerProps for the primary component and JsonNodeProps for the recursive child component.

Here's the prop table for the JsonViewer component:

| **Name** | **Type** | **Default** | **Description** |
| --- | --- | --- | --- |
| **jsonData** | unknown | - | Any valid JSON structure to be displayed |
| **initialExpanded** | boolean | false | Controls whether nodes are initially expanded or collapsed |
| **rootName** | string | false | "JSON" | Name for the root node. Use false to hide root name |

And here's the prop table for the internal JsonNode component:

| **Name** | **Type** | **Default** | **Description** |
| --- | --- | --- | --- |
| **nodeKey** | string | number | - | The key or index of the current node |
| **value** | unknown | - | The value to be displayed for this node |
| **level** | number | - | The nesting depth of the current node |
| **initiallyExpanded** | boolean | false | Controls whether this node starts expanded or collapsed |
