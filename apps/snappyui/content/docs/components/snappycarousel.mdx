## Props

| **Name**        | **Type**     | **Default** | **Description**|
|-----------------|--------------|-------------|----------------|
| items           | `Array`      | `[]`        | An array of items to be displayed in the carousel. Each item should have a `title`, `description`, and `image`. |
| variant         | `string`     | `horizontal`| The variant of the carousel. Can be `horizontal` or `vertical`. |
| autoPlay        | `boolean`    | `false`     | Whether the carousel should automatically play. |
| interval        | `number`     | `3000`      | The interval between slides in milliseconds. Only applicable if `autoPlay` is `true`. |
| className       | `string`     | `''`        | Additional classes to be applied to the carousel. |
| showArrows    | `boolean`    | `true`      | Whether to show the navigation arrows. |
| showDots      | `boolean`    | `true`      | Whether to show the navigation dots. |
