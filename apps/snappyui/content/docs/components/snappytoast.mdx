## Props

| Name        | Type                                                                 | Default              | Description                                             |
| ----------- | -------------------------------------------------------------------- | -------------------- | ------------------------------------------------------- |
| variant     | `"simple"` \| `"with-title"` \| `"with-action"`                      | `"simple"`           | Type of toast notification                              |
| title       | `string`                                                             | `-`                  | Toast title for "with-title" and "with-action" variants |
| description | `string`                                                             | `"This is a toast!"` | Description or message of the toast                     |
| action      | `React.ReactNode`                                                    | `-`                  | Action element like a button (only for with-action)     |
| duration    | `number`                                                             | `3000`               | Auto-dismiss time in milliseconds                       |
| position    | `"top-left"` \| `"top-right"` \| `"bottom-left"` \| `"bottom-right"` | `"bottom-right"`     | Placement on screen                                     |
| className   | `string`                                                             | `-`                  | Custom styles for toast wrapper                         |
| buttonText  | `string`                                                             | `"Show Toast"`       | Button Text                                             |
