### MovingBorder Component

The `MovingBorder` component can be used independently:

```tsx
import { MovingBorder } from "@/components/ui/snappy-button";

export default function CustomMovingBorder() {
  return (
    <div className="relative h-40 w-40">
      <MovingBorder duration={2000} rx="30%" ry="30%" borderColor="#22c55e">
        <div className="h-20 w-20 opacity-80" 
          style={{ background: "radial-gradient(#22c55e 40%, transparent 60%)" }} 
        />
      </MovingBorder>
      <div className="relative flex h-full w-full items-center justify-center">
        Your content here
      </div>
    </div>
  );
}
```

### Moving Border Animation

The moving border variant creates a dynamic animated border effect.

<div className="flex flex-col items-center justify-center mb-6 border border-zinc-800 rounded-lg py-5">
  <div className="flex flex-wrap justify-center gap-4">
    <Button 
      variant="moving-border" 
      borderColor="#0ea5e9"
    >
      Default Blue
    </Button>
    <Button 
      variant="moving-border" 
      borderColor="#22c55e"
    >
      Custom Green
    </Button>
    <Button 
      variant="moving-border" 
      borderColor="#ef4444"
      duration={2000}
    >
      Faster Red
    </Button>
  </div>
</div>

```tsx
<Button variant="moving-border" borderColor="#0ea5e9">
  Default Blue
</Button>

<Button variant="moving-border" borderColor="#22c55e">
  Custom Green
</Button>

<Button variant="moving-border" borderColor="#ef4444" duration={2000}>
  Faster Red
</Button>
```

You can also use a custom border style with Tailwind classes:

```tsx
<Button 
  variant="moving-border" 
  borderClassName="bg-gradient-to-r from-purple-500 to-pink-500"
>
  Gradient Border
</Button>
```

### Event Handling

```tsx

export default function ButtonWithEvent() {
  const handleClick = () => {
    alert("Button clicked!");
  };

  return (
    <Button onClick={handleClick} variant="primary">
      Click Me
    </Button>
  );
}
```

## Props

| Name             | Type            | Default     | Description                                     |
| ---------------- | --------------- | ----------- | ----------------------------------------------- |
| **variant**      | string          | 'default'   | Button style variant                            |
| **size**         | string          | 'default'   | Button size (sm, default, lg, xl)               |
| **disabled**     | boolean         | false       | Disables the button if true                     |
| **children**     | ReactNode       | -           | Button label or content                         |
| **onClick**      | Function        | -           | Function called when button is clicked          |
| **className**    | string          | -           | Extra Tailwind/custom classes (optional)        |
| **duration**     | number          | 3000        | Animation duration for moving-border in ms      |
| **borderRadius** | string          | "1.75rem"   | Border radius for button                        |
| **borderColor**  | string          | "#0ea5e9"   | Color for moving border (moving-border variant) |
| **borderClassName** | string       | -           | Custom Tailwind classes for border element      |

---

## Button Variants

| Variant          | Description                                   |
| ---------------- | --------------------------------------------- |
| **default**      | Standard button with dark/light mode styles   |
| **primary**      | Blue-themed button for primary actions        |
| **success**      | Green-themed button for success actions       |
| **destructive**  | Red-themed button for destructive actions     |
| **outline**      | Transparent button with border                |
| **ghost**        | Transparent button without border             |
| **warning**      | Yellow-themed warning button                  |
| **gradient**     | Button with gradient background               |
| **moving-border**| Button with animated moving border effect     |
