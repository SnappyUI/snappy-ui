
---

## Props

| Name         | Type      | Default   | Description                       |
| ------------ | --------- | --------- | --------------------------------- |
| **variant**  | string    | 'default' | Button style variant              |
| **size**     | string    | 'default' | Button size (sm, default, lg, xl) |
| **disabled** | boolean   | false     | Disables the button if true       |
| **children** | ReactNode | -         | Button label or content           |
| **onClick** | Function | -         | you can add custom funtion           |
