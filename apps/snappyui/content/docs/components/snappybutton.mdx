# Snappy Button Component

A flexible button component with multiple variants and sizes for different use cases.

---

## Installation

```bash
npm install @snappy/ui  # Install Snappy UI package
```

```tsx

</SnappyButton>
```

---

## Props

| Name        | Type           | Default   | Description                     |
|------------|---------------|-----------|---------------------------------|
| **variant**  | string       | 'default' | Button style variant            |
| **size**     | string       | 'default' | Button size (sm, default, lg, xl) |
| **disabled** | boolean      | false     | Disables the button if true     |
| **children** | ReactNode    | -         | Button label or content         |
