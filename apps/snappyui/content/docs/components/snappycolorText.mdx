### Installation

<Tabs items={["pnpm", "npm", "bun", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm dlx shadcn@latest add http://snappyui.in/api/registry/colorful-text
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npx shadcn@latest add http://snappyui.in/api/registry/colorful-text
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    npx shadcn@latest add http://snappyui.in/api/registry/colorful-text
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bunx --bun shadcn@latest add http://snappyui.in/api/registry/colorful-text
    ```
  </Tab>
</Tabs>

## Props

| **Name** | **Type** | **Description** |
| -------- | -------- | --------------- |
| text     | `string`   | You can highlight your text with a visual style (e.g., `SnappyText`, `ColourfulText`, `OceanText`, `SunsetText`, `NeonText`, `MonochromeText`, `ForestText`, `GalaxyText`, `CyberpunkText`, `VintageText`, `CandyText`, `MatrixText`, `PastelText`) |
