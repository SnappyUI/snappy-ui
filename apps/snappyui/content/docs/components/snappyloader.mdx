## Props

| **Name**   | **Type**                                 | **Default**   | **Description**                        |
| ---------- | ---------------------------------------- | ------------- | -------------------------------------- |
| `color`    | `string`                                 | `-`           | You can change color of the Loader     |
| `thicknees`| `"thin"` \| `"normal"` \| `"thick"`      | `-`           | Thickness of the Loader                |
| `speed`    | `"slow"` \| `"normal"` \| `"fast"`       | `-`           | Only for BarLoader                     |
| `size`     | `"small"` \| `"medium"` \| `"large"`     | `-`           | Size of Loader                         |
