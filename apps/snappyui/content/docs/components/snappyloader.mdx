# Snappy Loader Component

Snappy Loader is a simple and customizable loader component for React applications.

---

## Installation

<Tabs items={["pnpm", "npm", "bun", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm dlx shadcn@latest add http://dev.snappyui.in/api/registry/loader
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npx shadcn@latest add http://dev.snappyui.in/api/registry/loader
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    npx shadcn@latest add http://dev.snappyui.in/api/registry/loader
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bunx --bun shadcn@latest add http://dev.snappyui.in/api/registry/loader
    ```
  </Tab>
</Tabs>

## Props

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **color**     | string      | -| You can change color of the Loader |
| **thicknees**   | "thin" or | "normal" or "thick"     | Size of Loader                  |
| **speed**     | "slow" or "normal" or "fast"      | -              | Only for BarLoader                       |
| **size**      | "small" or "medium" or "large      | -             | Size of Loader                             |
