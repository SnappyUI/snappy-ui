# Snappy Loader Component

Snappy Loader is a simple and customizable loader component for React applications. It provides three types of loaders: DotLoader, BarLoader, and Loader. You can easily customize the color and size of the loaders to fit your application's design.

---

## Installation

<Tabs items={["pnpm", "npm", "bun", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm dlx shadcn@latest add http://dev.snappyui.in/api/registry/loader

  </Tab>
  <Tab value="npm">

  </Tab>
  <Tab value="yarn">

    npx shadcn@latest add http://dev.snappyui.in/api/registry/loader
    ```
  </Tab>
  <Tab value="bun">

## Props

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **color**     | string      | 'classic','primary','secondary','accent','string for coutome color(tailwind)'| You can change color of the Loader |
| **size**   | string      | 'small','medium','large'| Size of Loader                  |
| **progress**     | Number      | -              | Only for BarLoader                       |
