## Basic Usage

<Tabs
  items={[
    "Preview",
    "Code",
  ]}
>

    <Tab value="Preview">

      <div className="flex items-center justify-center">
      <TooltipProvider>
        <Tooltip>
            <TooltipTrigger>Hover me !</TooltipTrigger>
            <TooltipContent

            size="md"

## Props

| <b>Name</b>        | <b>Type</b>         | <b>Default</b>   | <b>Description</b>                              |
|------------|-------------|-----------|------------------------------------------|
| **variant**  | string      | default         | variant choose from default, info, warning, error, success                   |
| **size**  | string      | md         | size choose from sm, md, lg           |
| **className** | string    | -         | Additional CSS classes                  |
| **hasArrow** | boolean     | false        | HasArrow options from true or false            |
