## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| **holdDuration** | `number` | `1500` | Duration in milliseconds to hold before triggering |
| **label** | `string` | `"Hold to Delete"` | Text to display on the button |
| **deleteLabel** | `string` | `"Hold to Delete"` | Text to display during delete state |
| **className** | `string` | `""` | Custom class for the button |
| **deleteClassName** | `string` | `""` | Class for the delete state overlay |
| **icon** | `React.ReactNode` | `<DeleteIcon />` | Custom icon component |
| **onDelete** | `() => void` | ‚Äì | Function to call when hold is completed |
| **showProgress** | `boolean` | `true` | Show progress indicator during hold |
