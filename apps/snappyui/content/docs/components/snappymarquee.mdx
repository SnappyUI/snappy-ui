### Marquee Component Props

The Marquee component allows you to create horizontally scrolling text or content with various customization options.

| Prop           | Type                                                    | Default     | Description                                                       |
| ----------     | ------------------------------------------------------- | ----------- | ----------------------------------------------------------------- |
| `children`     | `React.ReactNode`                                       | *Required*  | The content to display inside the marquee                         |
| `variant`      | `"primary"` \| `"secondary"` \| `"accent"` \| `"muted"` | `"primary"` | Visual style of the marquee                                       |
| `speed`        | `"slow"` \| `"normal"` \| `"fast"`                      | `"normal"`  | Speed of the scrolling animation                                  |
| `direction`    | `"left"` \| `"right"`                                   | `"left"`    | Direction of the scrolling animation                              |
| `pauseOnHover` | `boolean`                                               | `true`      | Whether to pause the animation on hover                           |
| `draggable`    | `boolean`                                               | `false`     | Whether the marquee content can be dragged horizontally           |
| `className`    | `string`                                                | `""`        | Additional CSS classes to apply to the marquee container          |
| `darkMode`     | `boolean`                                               | `false`     | Whether to use dark mode color variants                           |
| `gap`          | `number`                                                | `16`        | Gap between repeated content sections in pixels                   |
| `repeat`       | `number`                                                | `3`         | Number of times the content should be repeated within the marquee |

### Variant Options

Each variant has different styling based on whether `darkMode` is enabled:

| Variant     | Light Mode                                       | Dark Mode                                       |
|------------ | ------------------------------------------------ | ----------------------------------------------- |
| `primary`   | Blue background with blue text                   | Dark blue background with light blue text       |
| `secondary` | Light gray background with dark gray text        | Dark gray background with light gray text       |
| `accent`    | Light purple background with purple text         | Dark purple background with light purple text   |
| `muted`     | Very light gray background with medium gray text | Very dark gray background with medium gray text |

### Speed Options

| Option   | Description                             |
|--------- | ----------------------------------------|
| `slow`   | Slow scrolling speed (40ms per pixel)   |
| `normal` | Medium scrolling speed (25ms per pixel) |
| `fast`   | Fast scrolling speed (15ms per pixel)   |
