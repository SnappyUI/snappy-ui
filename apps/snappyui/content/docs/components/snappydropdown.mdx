## Props

| Name         | Type      | Default   | Description                       |
| ------------ | --------- | --------- | --------------------------------- |
| **variant**  | "default" or  "outline" or "ghost" or  "link"  | 'default' | Style variant for trigger or button             |
| **inset**     | boolean    | false | Adds left padding to align with icons |
| **icon** | React.ReactNode   | -     | Optional icon to show before the content       |
| **iconRight** | React.ReactNode | -         | Optional icon to show on the right side           |
| **description** | string | -         | Additional descriptive text below the label           |
| **size** | "default" or "large" or "compact" | 'default'         | Padding size for DropdownMenuContent           |
| **badge** | React.ReactNode | -         | Optional badge element for labels           |
| **title** | string | -         | Title text in DropdownMenuHeader           |
