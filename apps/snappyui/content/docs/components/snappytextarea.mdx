## Props

| **Name** | **Type** | **Default** | **Required** | **Description** |
|------------|----------|-------------|--------------|-------------------------------------------------------------------------------|
| **id** | string   | -           | Yes          | The `id` attribute for the textarea element (for accessibility and linking). |
| **label** | string   | -           | Yes          | The text to display as the label for the textarea.                            |
| **placeholder** | string   | `''`        | No           | The placeholder text to display when the textarea is empty.                  |
| **rows** | number   | `4`         | No           | The initial number of visible text lines in the textarea.                     |
| **value** | string   | -           | No           | The current value of the textarea (for controlled components).                 |
| **onChange** | `(newValue: string) => void` | -           | No           | A callback function that is called when the textarea's value changes. It receives the new value as a string. |
| **className** | string   | `''`        | No           | Additional CSS classes to apply to the textarea container.                    |
