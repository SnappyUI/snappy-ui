## Props

| Name              | Type         | Default       | Required     | Description                                                             |
| ----------------- | ------------ | ------------- | ------------ | ----------------------------------------------------------------------- |
| title             | `string`     | `-`           | `Yes`        | The main heading text displayed in the hero section                     |
| description       | `string`     | `-`           | `Yes`        | Descriptive text that appears below the title                           |
| getStartedText    | `string`     | `-`           | `Yes`        | Text for the primary call-to-action button                              |
| learnMoreText     | `string`     | `-`           | `Yes`        | Text for the secondary action button                                    |
| imageUrl          | `string`     | `-`           | `Yes`        | URL of the image displayed in the hero section                          |
| imageAlt          | `string`     | `-`           | `Yes`        | Alternative text for the hero image for accessibility                   |
| className         | `string`     | `-`           | `No`         | Additional CSS classes to apply to the section container                |
| onGetStartedClick | `function`   | `-`           | `Yes`        | Callback `function` executed when the primary button is clicked         |
| onLearnMoreClick  | `function`   | `-`           | `Yes`        | Callback function executed when the secondary button is clicked         |
