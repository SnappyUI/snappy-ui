## Props

| Name               | Type                  | Default     | Description                                       |
| ------------------ | --------------------- | ----------- | ------------------------------------------------- |
| **value**          | `number`              | -           | Controlled value for the rating                   |
| **defaultValue**   | `number`              | `0`         | Default value for uncontrolled component          |
| **onChange**       | `(value: number) => void` | -       | Callback when rating changes                      |
| **count**          | `number`              | `5`         | Number of stars to display                        |
| **size**           | `'sm' \| 'default' \| 'lg'` | `'default'` | Size of the stars                          |
| **readOnly**       | `boolean`             | `false`     | Whether the rating can be changed                 |
| **precision**      | `1 \| 0.5`            | `1`         | Allow half-star ratings when set to 0.5           |
| **emptyIcon**      | `React.ReactNode`     | `'☆'`       | Icon for empty state                              |
| **filledIcon**     | `React.ReactNode`     | `'★'`       | Icon for filled state                             |
| **emptyColor**     | `string`              | -           | Color for empty stars                             |
| **filledColor**    | `string`              | -           | Color for filled stars                            |
| **hoverColor**     | `string`              | -           | Color for stars on hover                          |
| **className**      | `string`              | -           | Additional class for the container                |
| **iconClassName**  | `string`              | -           | Additional class for the icons                    |
| **animationStyle** | `'pulse' \| 'bounce' \| 'wobble' \| 'sparkle' \| 'flip' \| 'pop' \| 'none'` | `'none'` | Animation style when star is clicked |

---

## Animation Styles

| Style         | Description                                           |
| ------------- | ----------------------------------------------------- |
| **pulse**     | Stars smoothly scale up and down                      |
| **bounce**    | Stars bounce up and down                              |
| **wobble**    | Stars rotate back and forth                           |
| **sparkle**   | Stars emit a sparkling effect                         |
| **flip**      | Stars rotate around y-axis                            |
| **pop**       | Stars quickly contract and expand                     |
| **none**      | No animation applied                                  |
