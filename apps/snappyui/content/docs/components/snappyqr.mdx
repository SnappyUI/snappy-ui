## Props



### QRCode Component

| **Name**                | **Type**                   | **Default** | **Description**                                          |
| ----------------------- | -------------------------- | ----------- | -------------------------------------------------------- |
| **value**               | string                     | -           | The text/URL to encode in the QR code (required)         |
| **size**                | number                     | 200         | Size of the QR code in pixels (width and height)         |
| **bgColor**             | string                     | '#FFFFFF'   | Background color in hex format                           |
| **fgColor**             | string                     | '#000000'   | Foreground color in hex format (QR code color)           |
| **errorCorrectionLevel**| 'L' | 'M' | 'Q' | 'H'  | 'M'         | Error correction level (L:7%, M:15%, Q:25%, H:30% recovery) |
| **includeMargin**       | boolean                    | true        | Whether to include a margin/quiet zone around the QR code|
