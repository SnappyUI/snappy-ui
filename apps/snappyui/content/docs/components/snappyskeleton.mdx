## Props

| Name          | Type       | Default        | Description                                              |
| ------------- | ---------- | -------------- | -------------------------------------------------------- |
| **variant**   | `string`   | `rectangular`  | Choose from 'text', 'circular', "rectangular", "rounded" |
| **className** | `string`   | `-`            | Additional CSS classes                                   |
| **animation** | `string`   | `pulse`        | Options: "pulse" or false                                |
| **width**     | `number`   | `100%`         | Custom width                                             |
| **height**    | `number`   | `-`            | Custom height                                            |
