## Props

| <b>Name</b>        | <b>Type</b>         | <b>Default</b>   | <b>Description</b>                              |
|------------|-------------|-----------|------------------------------------------|
| **variant**  | string      | rectangular         | variant choose from text, circular, rectangular, rounded                   |
| **className** | string    | -         | Additional CSS classes                  |
| **animation** | string     | pulse        | Animation options from pulse or false            |
| **width**  | number      | 100%        | Custom width                 |
| **height** | number     | - | Custom Height          |
