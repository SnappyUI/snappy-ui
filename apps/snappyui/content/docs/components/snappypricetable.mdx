## Props

| <b>Name</b>        | <b>Type</b>         | <b>Default</b>   | <b>Description</b>                              |
|------------|-------------|-----------|------------------------------------------|
| **mostPopular**  | boolean      | false         | Highlights the card as "Most Popular" if true                   |
| **className** | string    | -         | Additional CSS classes                  |
| **title** | string     | -        | The  title of the pricing card/Feature            |
| **price**  | string      | -        | Price value (e.g., "$29")                |
| **disable** | boolean     | false | If true, feature is marked as unavailable       |
| **onClick** | clickEventFunction     | - | Click handler function for the button       |
