## Basic Usage

The toggle component allows users to switch between two states. By default, it uses the "simple" variant.

<Tabs
  items={[
    "Preview",
    "Code",
  ]}
>
  <Tab value="Preview">
    <ToggleExample variant="simple" />
  </Tab>

  <Tab value="Code">
  ```tsx
  import type { ToggleVariant } from "@/components/ui/snappy-toggle";

  <Toggle 
    variant="simple"
    checked={checked}
    onChange={setChecked}
  />
  ```
  </Tab>
</Tabs>

### Props

| Prop           | Type                            | Default     | Description                                                  |
| -------------- | ------------------------------- | ----------- | ------------------------------------------------------------ |
| variant        | `string`                        | `"simple"`  | Toggle style variant                                         |
| size           | `"sm"` \| `"default"` \| `"lg"` | `"default"` | Size of the toggle                                           |
| checked        | `boolean`                       | `false`     | Controlled checked state                                     |
| defaultChecked | `boolean`                       | `false`     | Initial checked state (uncontrolled)                         | 
| onChange       | `(checked: boolean) => void`    | `—`         | Callback when toggle state changes                           |
| disabled       | `boolean`                       | `false`     | Whether the toggle is disabled                               |
| onLabel        | `string`                        | `—`         | Label for the "on" state (for variants that support labels)  |
| offLabel       | `string`                        | `—`         | Label for the "off" state (for variants that support labels) |
| children       | `ReactNode`                     | `—`         | Optional child elements                                      |
