## Stacked Avatars

Use the `stacked` prop to create an overlapping avatar layout, useful for showing groups of users or collaborators.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <div className="flex items-center justify-center">
      <Avatar name="Alice" src="https://i.pravatar.cc/56?img=3" stacked />
      <Avatar name="Bob" src="https://i.pravatar.cc/56?img=4" stacked />
      <Avatar name="Charlie" src="https://i.pravatar.cc/56?img=5" stacked />
    </div>
  </Tab>

  <Tab value="Code">
    ```tsx
    import Avatar from "@/components/ui/snappy-avatar";

    <Avatar name="Alice" src="https://i.pravatar.cc/56?img=3" stacked />
    <Avatar name="Bob" src="https://i.pravatar.cc/56?img=4" stacked />
    <Avatar name="Charlie" src="https://i.pravatar.cc/56?img=5" stacked />
    ```
  </Tab>
</Tabs>

## Props

| Name    | Type      | Default | Description                                      |
| ------- | --------- | ------- | ------------------------------------------------ |
| name    | `string`  | `-`     | Name shown in tooltip and for accessibility      |
| src     | `string`  | `-`     | Image source URL                                 |
| width   | `number`  | `56`    | Width of the avatar in pixels                    |
| height  | `number`  | `56`    | Height of the avatar in pixels                   |
| stacked | `boolean` | `false` | Whether to render the avatar in a stacked layout |
