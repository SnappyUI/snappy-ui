## Props

| <b>Name</b>   | <b>Type</b> | <b>Default</b> | <b>Description</b>                            |
| ------------- | ----------- | -------------- | --------------------------------------------- |
| **title**     | string      | -              | Title of accordion                        |
| **children** | ReactNode      | -              | inside of accordion                        |
| **variant**   | "default" or "bordered" or "shadowed" or "minimal" or "glassmorphic" or "gradient" or "elevated"| "default" | variants of accordion |
| **isOpen**     | boolean      | false             | Set default state|
| **className**   | string      | -      | you can pass tailwind classes|
