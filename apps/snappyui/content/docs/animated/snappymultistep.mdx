## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `steps` | `MultiStepContent[]` | Required | Array of step objects with title, description, and skeleton widths |
| `initialStep` | `number` | `0` | Index of the initial step to display |
| `backButtonText` | `string` | `"Back"` | Text displayed in the back button |
| `continueButtonText` | `string` | `"Continue"` | Text displayed in the continue button |
| `doneButtonText` | `string` | `"Done"` | Text displayed in the final step button |
| `buttonSize` | `"sm" \| "md" \| "lg"` | `"md"` | Size of the navigation buttons |
| `backButtonStyle` | `ButtonStyle` | `{}` | Custom styling for the back button |
| `continueButtonStyle` | `ButtonStyle` | `{}` | Custom styling for the continue/done button |
| `onComplete` | `() => void` | `undefined` | Function called when the final step button is clicked |

### MultiStepContent Type

```tsx
type MultiStepContent = {
  title: string;
  description: string;
  skeletonWidths?: number[];
};
```

### ButtonStyle Type

```tsx
type ButtonStyle = {
  backgroundColor?: string;
  textColor?: string;
  hoverBackgroundColor?: string;
  hoverTextColor?: string;
  borderColor?: string;
};
```
