## Props

| Name                   | Type                         | Default       | Description                                     |
| ---------------------- | ---------------------------- | ------------- | ----------------------------------------------- |
| **initialDate**        | Date                         | -             | Pre-selected date when component renders        |
| **onChange**           | (date: Date \| null) => void | -             | Function called when date selection changes     |
| **placeholder**        | string                       | 'Select date' | Placeholder text for the input field            |
| **format**             | string                       | 'MM/DD/YYYY'  | Date format pattern for display                 |
| **label**              | string                       | -             | Label text displayed above the input field      |
| **showClearButton**    | boolean                      | true          | Shows the clear button if true                  |
| **showTodayButton**    | boolean                      | true          | Shows the today button if true                  |
| **disabled**           | boolean                      | false         | Disables the date picker if true                |
| **className**          | string                       | -             | Classes for the wrapper element                 |
| **containerClassName** | string                       | -             | Classes for the input container                 |
| **inputClassName**     | string                       | -             | Classes for the input field                     |
| **calendarClassName**  | string                       | -             | Classes for the calendar dropdown               |
| **todayButtonClassName**| string                      | -             | Classes for the Today button                    |
| **clearButtonClassName**| string                      | -             | Classes for the Clear button                    |
